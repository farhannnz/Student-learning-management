CDN Fix Instructions

Option 1: Disable Tracking Prevention (Quick Fix)

  1. In Edge browser, click the shield icon in address bar
  2. Turn OFF "Tracking prevention" for this site
  3. Refresh the page

Option 2: Use Local Supabase (Permanent Fix)

Download Supabase JS file locally:

  1. Go to: Download Supabase
  2. Save as "js/supabase.js"
  3. Update HTML files to use local file

Option 3: Use Different Browser

  • Try Chrome or Firefox
  • These browsers don't block CDN by default
